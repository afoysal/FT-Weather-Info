Weather Info displays weather information on your WordPress Website.

## Description ##

Weather Info is a simple solution to show weather information on WordPress Website. 

Weather Info collects information from OpenWeatherMap through API. OpenWeatherMap API key is required to use this Plugin.

## Features ##

** Show Weather Information for a city.
** Show Title.
** Show City name & Country name.
** Show Temperature.
** Show Current Time.
** Show Current Date.
** Show Wind Speed.
** Show Humidity Percentage.
** Show Pressure.
** Show Visibility Distance.
** Show Cloudiness Percentage.
** Show Sunrise time.
** Show Sunset time.
** Change Background color.
** Change Text color.
** Change font size of the Text.
** Change Temperature unit
** Show weather information using multiple languages.
** Easy to install 
** Need OpenWeatherMap API key


## Installation ##


## MINIMUM REQUIREMENTS ##

** PHP 5.6 or greater is recommended
** MySQL 5.6 or greater is recommended
** WordPress 4.5 or greater is recommended


How to install Weather Info plugin and get it working.


1. Download the plugin.
2. Connect to your FTP.
3. Upload the unzipped downloaded folder in the root plugins directory “/wp-content/plugins/”.
4. Activate the plugin from the Plugins menu of your admin.
5. If all went well, a new "Weather Info" menu should appear.
6. Get your free OpenWeatherMap API key at https://www.openweathermap.com and put it on Dashboard->Weather Info->API Key.


The WordPress codex contains [instructions on how to do this here](https://wordpress.org/support/article/managing-plugins/#manual-plugin-installation).




## Frequently Asked Questions ##




= What is Weather Info ? =

Weather Info is a plugin for WordPress which enables the user/users of a WordPress website to show Weather Information on their WordPress Website.


= How does it work ? =

Weather Info fetches weather information from https://www.openweathermap.com through API and show at WordPress Website.


= Is It Dependent On Anything Else ? =

Yes. API Key of https://www.openweathermap.com needed to use this Plugin.


= Is it possible to add multiples weathers in a same page ? =

Yes. Each weather can have a completely different setup and layout.


= What is the source of the data ? =

Open Weather Map (https://openwweathermap.com).


= How to define the unit Celsius or Fahrenheit? =

When creating your weather, choose "Celsius" or "Fahrenheit" from the drop down list "Temperature".


= How to define a custom title for my weather? =

Simply enter text in the "Add title".
